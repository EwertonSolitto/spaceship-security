# Installation 🔧

Make sure to have **NodeJS** and **Expo** installed.

```bash

# Clone repository
git clone https://github.com/EwertonSolitto/spaceship-security.git

# Go to the directory
cd spaceship-security

# Install dependencies
npm i

# Start Application
npm start

```
